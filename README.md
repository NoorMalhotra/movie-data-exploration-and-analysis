# Movie Data Analysis using Python

## Overview
This project focuses on analyzing a movie dataset using Python to uncover patterns related to ratings, budget, revenue, and genres.

This project was completed under the guidance of Mr. Kushal Sharma (EY Trainer & Guest Lecturer).

## Objectives
- Clean and preprocess raw movie data  
- Handle missing values and inconsistent formats  
- Explore relationships between budget, gross revenue, and IMDB ratings  
- Identify trends across genres, directors, and release years  

## Key Insights
- Drama is the most frequent genre in the dataset  
- Average IMDB rating for movies released after 2010 is approximately 6.24  
- A significant portion of data contains missing values, especially in budget and gross columns  
- Revenue distribution is highly skewed, with a small number of movies generating most of the total revenue  

## Tools Used
- Python (Pandas, Matplotlib)  

## Learnings
- Data cleaning is a crucial step before performing any analysis  
- Handling missing values and data types directly impacts accuracy  
- Properly structured data leads to more meaningful insights  

## Project Structure
- notebook/ : Jupyter notebook containing the full analysis  
- data/ : dataset used for analysis  
- support/ : Excel file used for additional data handling  

## Files

## Conclusion
This project helped in understanding the importance of data preprocessing and how clean, structured data improves the reliability of analysis.
